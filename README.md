# Java Calculator

Java Calculator is a simple and user-friendly application that performs basic arithmetic operations. The project is developed using Java and demonstrates the use of 
Object-Oriented Programming concepts and logical implementation.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear and reset functionality
- Simple and easy-to-use interface

## Tech Stack
- Java
- Object-Oriented Programming (OOP)
- Eclipse IDE

## Project Structure
miniProjects
│
├── src
└── calculator.java


## How to Run
1. Clone or download this repository
2. Open the project in Eclipse
3. Go to the `src` folder
4. Run the calculator.java file
5. Use the calculator to perform operations

## Learning Outcomes
- Understanding of Java basics
- Implementation of arithmetic logic
- Use of OOP concepts
- Handling user input and events
